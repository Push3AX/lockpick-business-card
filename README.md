# Lockpick Business Card

A lockpick business card inspired by [Kevin Mitnick](https://www.mitnicksecurity.com/kevin-mitnicks-famous-lockpick-business-card).

Includes a detachable lockpick set: hook pick, torsion wrench, half-diamond pick, snake rake, and ball pick.

一张受 [Kevin Mitnick](https://www.mitnicksecurity.com/kevin-mitnicks-famous-lockpick-business-card) 启发的开锁工具名片。

包含一套可拆式开锁工具：钩形单钩、张力扳手、半菱形单钩、蛇形耙和球形单钩。
